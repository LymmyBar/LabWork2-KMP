# Лабораторна робота №2: Комп'ютерне моделювання процесів

## Тема
Рух тіла, кинутого під кутом α до горизонту з початковою швидкістю v₀ з висоти h₀.

## Опис
Ця комп'ютерна модель реалізує математичну модель з ЛР1 для дослідження балістичного руху тіла.

### Математична модель
- **Система координат:** Ox — горизонтальна, Oy — вертикальна вгору.
- **Вхідні параметри:**
  - `v₀` — початкова швидкість (м/с), діапазон 10...100 м/с
  - `α` — кут кидання (градуси), діапазон 0°...90°
  - `h₀` — початкова висота (м), діапазон 0...50 м
  - `g = 9.81 м/с²` — прискорення вільного падіння

### Формули
- **Проекції швидкості:**
  - `v₀ₓ = v₀ · cos(α)`
  - `v₀ᵧ = v₀ · sin(α)`

- **Рівняння руху:**
  - `x(t) = v₀ · cos(α) · t`
  - `y(t) = h₀ + v₀ · sin(α) · t - (g · t²)/2`

- **Час підйому:** `tup = v₀ · sin(α) / g`

- **Максимальна висота:** `Hmax = h₀ + (v₀² · sin²(α)) / (2g)`

- **Час польоту:** `T = (v₀ · sin(α) + √(v₀² · sin²(α) + 2 · g · h₀)) / g`

- **Дальність польоту:** `L = v₀ · cos(α) · T`
