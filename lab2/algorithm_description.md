# Алгоритм роботи комп'ютерної моделі

## Опис алгоритму у вигляді нумерованих кроків

### Блок 1: Ініціалізація

1. **ПОЧАТОК**
2. Задати константу прискорення вільного падіння g = 9.81 м/с²
3. Задати діапазони вхідних параметрів:
   - v₀: 10...100 м/с
   - α: 0°...90°
   - h₀: 0...50 м

### Блок 2: Введення даних

4. Ввести початкову швидкість v₀ (м/с)
5. Ввести кут кидання α (градуси)
6. Ввести початкову висоту h₀ (м)
7. Перевірити коректність введених даних:
   - ЯКЩО v₀ < 10 АБО v₀ > 100, ТО вивести помилку, перейти до кроку 4
   - ЯКЩО α < 0 АБО α > 90, ТО вивести помилку, перейти до кроку 5
   - ЯКЩО h₀ < 0 АБО h₀ > 50, ТО вивести помилку, перейти до кроку 6

### Блок 3: Попередні обчислення

8. Перетворити кут α з градусів у радіани: α_rad = α × π / 180
9. Обчислити горизонтальну складову швидкості: v₀ₓ = v₀ × cos(α_rad)
10. Обчислити вертикальну складову швидкості: v₀ᵧ = v₀ × sin(α_rad)

### Блок 4: Обчислення характеристик польоту

11. Обчислити час підйому до максимальної висоти:
    ```
    tup = v₀ᵧ / g
    ```

12. Обчислити максимальну висоту підйому:
    ```
    Hmax = h₀ + (v₀ᵧ)² / (2 × g)
    ```

13. Обчислити дискримінант квадратного рівняння для часу польоту:
    ```
    D = (v₀ᵧ)² + 2 × g × h₀
    ```

14. Обчислити повний час польоту:
    ```
    T = (v₀ᵧ + √D) / g
    ```

15. Обчислити дальність польоту:
    ```
    L = v₀ₓ × T
    ```

### Блок 5: Генерація траєкторії

16. Задати кількість точок траєкторії N (наприклад, N = 100)
17. Обчислити крок часу: Δt = T / (N - 1)
18. ДЛЯ i = 0 ДО N-1:
    - Обчислити поточний час: t = i × Δt
    - Обчислити x-координату: x[i] = v₀ₓ × t
    - Обчислити y-координату: y[i] = h₀ + v₀ᵧ × t - g × t² / 2
    - ЯКЩО y[i] < 0, ТО y[i] = 0

### Блок 6: Виведення результатів

19. Вивести таблицю результатів:
    - Час польоту T
    - Максимальна висота Hmax
    - Дальність польоту L
    - Час підйому tup

20. Побудувати графік траєкторії y(x)
21. Додати підписи осей та легенду
22. Зберегти графік у файл

### Блок 7: Серія експериментів

23. ЯКЩО потрібна серія експериментів:
    - Задати масив кутів: angles = [α₁, α₂, ..., αₙ]
    - ДЛЯ кожного αᵢ з масиву angles:
      - Виконати кроки 8-15
      - Зберегти результати
    - Побудувати сімейство траєкторій на одному графіку

### Блок 8: Верифікація моделі

24. Порівняти числові результати з аналітичними формулами
25. Обчислити відносну похибку:

## Пояснення до ключових формул

### Час польоту T
Отримується як додатній розв'язок квадратного рівняння:
```
h₀ + v₀ᵧ·t - g·t²/2 = 0
```
Застосовуючи формулу квадратного рівняння:
```
t = (-v₀ᵧ ± √(v₀ᵧ² + 2·g·h₀)) / (-g)
T = (v₀ᵧ + √(v₀ᵧ² + 2·g·h₀)) / g  (додатній корінь)
```

### Максимальна висота Hmax
У точці максимальної висоти вертикальна швидкість дорівнює нулю:
```
vᵧ(tup) = v₀ᵧ - g·tup = 0
tup = v₀ᵧ / g
Hmax = y(tup) = h₀ + v₀ᵧ·tup - g·tup²/2 = h₀ + v₀ᵧ²/(2g)
```

### Дальність польоту L
Горизонтальна відстань за час T:
```
L = x(T) = v₀ₓ · T = v₀·cos(α)·T
```
