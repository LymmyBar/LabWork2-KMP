# Блок-схема алгоритму моделі польоту тіла

```mermaid
flowchart TD
    A[Початок] --> B[Введення вихідних даних
v0, h0, список кутів α]
    B --> C[Для кожного h0]
    C --> D[Для кожного α]
    D --> E[Переведення α з градусів у радіани]
    E --> F[Обчислення складових швидкості
v0x = v0·cosα
v0y = v0·sinα]
    F --> G[Обчислення часу підйому
t_up = v0y / g]
    G --> H[Обчислення максимальної висоти
Hmax = h0 + v0²·sin²α / (2g)]
    H --> I[Обчислення дискримінанта
d = v0²·sin²α + 2gh0]
    I --> J[Обчислення часу польоту
T = (v0y + √d) / g]
    J --> K[Обчислення дальності польоту
L = v0x · T]
    K --> L[Збереження результатів
(v0, h0, α, T, L, Hmax)]
    L --> M{Є ще α?}
    M -- Так --> D
    M -- Ні --> N{Є ще h0?}
    N -- Так --> C
    N -- Ні --> O[Виведення підсумкової таблиці
та побудова графіків]
    O --> P[Перевірка адекватності моделі
(тестовий приклад)]
    P --> Q[Кінець]
```

Цю блок-схему GitHub автоматично відобразить як діаграму Mermaid у файлі `flowchart.md`. Для перегляду просто відкрий цей файл у репозиторії.
