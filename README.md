# Лабораторна робота №2: Комп'ютерне моделювання процесів

## Тема
Комп'ютерна модель руху тіла, кинутого під кутом α до горизонту з початковою швидкістю v₀ з висоти h₀ в полі тяжіння Землі без опору повітря.

## Опис

Ця програма реалізує математичну модель руху тіла, кинутого під кутом до горизонту, та дозволяє:

- Обчислювати характеристики польоту (дальність L, час T, максимальна висота H_max)
- Досліджувати залежність параметрів польоту від початкових умов
- Знаходити оптимальний кут для максимальної дальності
- Візуалізувати траєкторії та залежності
- Верифікувати модель порівнянням з теоретичними формулами

## Математична модель

### Вхідні параметри
- `v₀` — початкова швидкість (м/с), діапазон: 10..100
- `α` — кут кидання (градуси), діапазон: 0..90
- `h₀` — початкова висота (м), діапазон: 0..50
- `g` = 9.81 м/с² — прискорення вільного падіння

### Формули
- Компоненти швидкості: `v₀x = v₀ · cos(α)`, `v₀y = v₀ · sin(α)`
- Координати: `x(t) = v₀ · cos(α) · t`, `y(t) = h₀ + v₀ · sin(α) · t - (g · t²)/2`
- Час підйому: `t_up = v₀ · sin(α) / g`
- Максимальна висота: `H_max = h₀ + v₀² · sin²(α) / (2g)`
- Час польоту: `T = (v₀ · sin(α) + √(v₀² · sin²(α) + 2·g·h₀)) / g`
- Дальність: `L = v₀ · cos(α) · T`


1. Аналіз залежності параметрів від кута кидання
2. Пошук оптимального кута для різних висот
3. Побудова графіків траєкторій
4. Побудова графіків залежностей
5. Верифікація моделі

## Ключові висновки

- При h₀ = 0 оптимальний кут для максимальної дальності ≈ 45°
- При h₀ > 0 оптимальний кут < 45°
- Зі збільшенням h₀ оптимальний кут зменшується
